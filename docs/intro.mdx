# Introduction 👋

<p className="text-justify">
  QuranDB adalah sebuah aplikasi yang dibuat untuk memudahkan pengguna dalam
  mencari <span className="font-bold">repetisi</span> ayat Al-Qur'an berdasarkan
  kata kunci. Selain itu, aplikasi ini juga dapat digunakan untuk mencari{" "}
  <span className="font-bold">terjemahan</span>,{" "}
  <span className="font-bold">transliterasi</span>, serta{" "}
  <span className="font-bold">relasi</span> antar ayat Al-Qur'an. Dibuat
  menggunakan teknologi Javascript yang memungkinkan aplikasi ini dapat diakses
  secara <span className="font-italic">online</span> maupun{" "}
  <span className="font-italic">offline</span> dan tersedia di berbagai
  perangkat.
</p>

## Features 📋

- Pencarian 🔍
  - Ayat Al-Qur'an berdasarkan kata kunci (Arab atau Indonesia)
  - Akar kata Bahasa Arab - On Progress 🚧
- Relasi 🔁
  - Ayat Al-Qur'an dengan terjemahan Bahasa Indonesia
  - Ayat Al-Qur'an dengan transliterasi Bahasa Indonesia
- Menampilkan repetisi ayat Al-Qur'an berdasarkan kata kunci ♾️
- Terjemah per kata (kata perkata) Bahasa Indonesia 🌐 - On Progress 🚧

### Additional Features 📋

- Terintegrasi dengan AI (Artificial Intelligence) 🤖 - On Progress 🚧
- Tampilan yang responsif dan mudah digunakan 📱
- Mode malam 🌓
- Print hasil pencarian 🖨️
- Export hasil pencarian ke PDF 📄
- Export hasil pencarian ke Excel 📗
- Menampilkan ringkasan kata kunci yang dicari 📖
